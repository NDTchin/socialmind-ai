# 🚀 SocialMind AI - Facebook Auto-Posting Tool

## 📝 Introduction

**SocialMind AI** allows you to use multiple Facebook accounts to automatically post content to various groups.

## 📦 Technologies Used

- 🖥️ **Selenium**: Automates operations on Chrome Driver.
- 📊 **Numpy, Pandas**: Data reading and processing.
- 🖱️ **PyAutoGUI**: Automatically selects images from library for posting.

## 🔧 Usage Guide

### 1️⃣ Clone the project
```bash
git clone https://github.com/NDTchin/socialmind-ai.git
```

### 2️⃣ Install dependencies
```bash
pip install -r requirements.txt
```

### 3️⃣ Configuration
- Customize paths, images, and post content.
- Create `accounts.csv` file in `/data` folder as needed.

### 4️⃣ Run the project
```bash
python main_posts.py
```
